## ✨ Features

- 🔍 **Real-Time Search** — Instant product filtering as you type
- 🗂️ **Category Filters** — Browse by Clothing, Accessories, Home & Living, New, or Sale
- 🛒 **Slide-In Cart Drawer** — Add, remove, and adjust quantities with live subtotal
- 🪟 **Product Detail Modal** — Full product info with add-to-cart
- ❤️ **Wishlist Toggle** — Save favourite products
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop
- 🍞 **Toast Notifications** — Feedback on every cart interaction
- 📧 **Newsletter Signup** — Subscriber capture form
